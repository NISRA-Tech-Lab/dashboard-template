Information about this dashboard

This dashboard shows total greenhouse gas emissions for Northern Ireland over time. It allows users to see long term trends since 1990 and how emissions have changed in the most recent years.

All figures come from the UK Greenhouse Gas Inventory and represent emissions occurring within Northern Ireland’s territory.

More information can be found on the User Notes page.
